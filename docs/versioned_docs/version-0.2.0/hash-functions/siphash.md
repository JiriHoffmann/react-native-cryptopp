# SipHash

```js
Cryptopp.hashFunctions.SipHash(data, '4_8_128');
```

#### Parameters

| Parameter | Type                         | Required |
| --------- | ---------------------------- | -------- |
| data      | `string` <br/> `ArrayBuffer` | ✅       |
| type      | `"2_4_64"` <br/> `"4_8_128"` | ✅       |

#### Returns: `string`
