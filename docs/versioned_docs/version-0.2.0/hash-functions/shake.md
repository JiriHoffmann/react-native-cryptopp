# SHAKE

```js
Cryptopp.hashFunctions.SHAKE(data, '256');
```

#### Parameters

| Parameter | Type                         | Required |
| --------- | ---------------------------- | -------- |
| data      | `string` <br/> `ArrayBuffer` | ✅       |
| size      | `"128"` <br/> `"256"`        | ✅       |

#### Returns: `string`
