# RIPEMD

```js
Cryptopp.hashFunctions.RIPEMD(data, '256');
```

#### Parameters

| Parameter | Type                                              | Required |
| --------- | ------------------------------------------------- | -------- |
| data      | `string` <br/> `ArrayBuffer`                      | ✅       |
| size      | `"128"` <br/> `"160"` <br/> `"256"` <br/> `"320"` | ✅       |

#### Returns: `string`
