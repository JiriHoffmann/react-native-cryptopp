# LSH

```js
Cryptopp.hashFunctions.LSH(data, '512');
```

#### Parameters

| Parameter | Type                                              | Required |
| --------- | ------------------------------------------------- | -------- |
| data      | `string` <br/> `ArrayBuffer`                      | ✅       |
| size      | `"224"` <br/> `"256"` <br/> `"384"` <br/> `"512"` | ✅       |

#### Returns: `string`
