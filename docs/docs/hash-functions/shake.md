# SHAKE

```js
Cryptopp.hashFunctions.SHAKE(data);
```

#### Parameters

| Parameter | Type                      | Required |
| --------- | ------------------------- | -------- |
| data      | `string` <br/> `ArrayBuffer`  | ✅       |
| size      | `"128"`  <br/> `"256"`        | ✅       |

#### Returns: `string`
